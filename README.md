# Deepak Patel Portfolio

This repository contains my personal portfolio website and projects developed during my Computer Science journey.

## Featured Projects

### Smart Grid Anti-Theft Monitoring System

A full-stack + embedded systems project using ESP32, Streamlit dashboards, LCD display, LEDs, and simulated smart-grid telemetry to detect and visualize electricity theft events.

### Scientific Calculator

Built using HTML, CSS, and JavaScript with custom expression parsing and scientific functions.

### To-Do List Manager

Task management application with localStorage persistence and filtering functionality.

### Notepad Application

Browser-based text editor with undo/redo, live statistics, and file handling support.

## AI-Assisted Development

This project was developed using AI-assisted rapid prototyping with tools such as ChatGPT and Google Gemini for debugging support, UI refinement, architecture brainstorming, and workflow optimization.

The system integration, hardware setup, dashboard customization, project logic, and final implementation were completed and modified manually during development.
